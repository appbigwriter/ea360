## Story

**As a** empreendedor que investe em canais de monetização,
**I want** registrar manualmente as métricas reais de cada canal (ROAS, CPA, payback, contribuição) para um período,
**so that** o Painel 360 possa comparar o real com o projetado.

## Acceptance Criteria

1. Formulário em `/app/panel/metrics` permite inserir `spend_actual`, `roas`, `cpa`, `payback_actual`, `contribution` por canal para um período (data início/fim).
2. Dados submetidos são salvos em `channel_metrics` via Server Action.
3. Validação: `spend_actual >= 0`, datas válidas, `period_end >= period_start`.
4. Múltiplos registros de métrica para o mesmo canal/período são permitidos (histórico).
5. Lista de métricas inseridas exibida abaixo do formulário, ordenada por data de criação descendente.
6. Para MVP, ingest é manual (sem conectores automáticos — roadmap §4).
