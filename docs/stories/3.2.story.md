## Story

**As a** empreendedor autenticado,
**I want** uma interface conversacional para responder à Entrevista 360 passo a passo,
**so that** possa fornecer as informações sobre meu negócio de forma natural e guiada.

## Acceptance Criteria

1. Rota autenticada `/app/interview` exibe a interface conversacional da Entrevista 360.
2. Interface exibe uma pergunta por vez com progresso visual das quatro camadas.
3. Usuário pode responder por texto livre.
4. Botão "Próxima" avança para a próxima pergunta; botão "Anterior" permite revisitar.
5. Indicador de camada atual visível (ex: "Camada 2 de 4: Filosofia & Valores").
6. Interface responsiva (mobile + desktop).
7. Estado de carregamento exibido enquanto perguntas são geradas/carregadas.
8. Entrevista nova cria registro em `interviews` com status `in_progress` ao iniciar.
