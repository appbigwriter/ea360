## Story

**As a** visitante do EA360 (sem conta),
**I want** navegar, buscar e filtrar os canais de monetização do GOM sem precisar fazer login,
**so that** possa conhecer as opções disponíveis antes de criar uma conta.

## Acceptance Criteria

1. Rota pública `/gom` exibe lista de todos os canais, agrupados por pilar.
2. Busca por nome de canal funciona em tempo real (debounce ~300ms) sem reload de página.
3. Filtro por pilar (Ads / Afiliações / Parcerias) funciona como tabs ou chips.
4. Filtro por atributo (ex: ordenar por Custo crescente) disponível.
5. Command palette (`cmdk`) acessível via atalho de teclado (Ctrl+K / Cmd+K) para busca rápida de canal.
6. Página funciona sem autenticação (consulta ao Supabase com anon key + RLS público).
7. Layout responsivo (mobile + desktop).
8. Estado de carregamento (skeleton) exibido enquanto dados são carregados.
