## Story

**As a** sistema EA360,
**I want** ingerir documentos de política da Meta (páginas scrapeadas) e gerar embeddings para armazená-los no Oráculo,
**so that** o Q&A RAG possa encontrar trechos relevantes ao responder perguntas de conformidade.

## Acceptance Criteria

1. Script/função de ingestão aceita `{title, content, url, source_type}` e gera embedding via API OpenAI (`text-embedding-3-small`).
2. Embedding gerado é salvo em `oracle_documents.embedding` junto com o conteúdo.
3. Textos longos são divididos em chunks de ~500 tokens com overlap de ~50 tokens antes de embedar.
4. Ingestão inicial popula ao menos 5 documentos de política da Meta de exemplo.
5. Chave `OPENAI_API_KEY` usada apenas no servidor; nunca exposta no cliente.
6. Função de ingestão é chamável via script Node.js para operações de manutenção.
