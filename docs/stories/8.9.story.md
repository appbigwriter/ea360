## Story

**As a** empreendedor com funil e templates configurados,
**I want** exportar o fluxo do funil como um workflow n8n importável,
**so that** possa executar o funil de WhatsApp no n8n sem precisar reconfigurar manualmente cada nó.

## Acceptance Criteria

1. Botão "Exportar para n8n" em `/app/executor/funnel` gera um arquivo JSON de workflow n8n.
2. JSON gerado contém nós n8n correspondentes a cada etapa do funil: trigger webhook, envio de template, lógica de escape humano.
3. Arquivo JSON é baixado pelo browser como `funnel-{businessName}-{date}.json`.
4. JSON é compatível com importação nativa do n8n (formato de workflow n8n).
5. Templates de mensagem gerados na Forja (Story 8.2) são referenciados nos nós do workflow.
6. Prioridade Should: funcionalidade valiosa mas não bloqueia outros stories do épico.
