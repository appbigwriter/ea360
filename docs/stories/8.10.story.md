## Story

**As a** empreendedor com conta Meta Business verificada,
**I want** conectar o EA360 à Meta Marketing API e WhatsApp Cloud API,
**so that** possa criar campanhas CTWA e enviar templates WhatsApp diretamente pela plataforma.

## Acceptance Criteria

1. Configuração de credenciais Meta em `/app/settings/integrations`: `WABA_ID`, `META_ACCESS_TOKEN`, `PHONE_NUMBER_ID`.
2. Credenciais salvas criptografadas (nunca em texto plano no banco).
3. Botão "Testar conexão" verifica se credenciais são válidas via endpoint Meta Graph API.
4. Envio de template WhatsApp via `POST /v18.0/{WABA_ID}/messages` com template aprovado.
5. Criação de campanha CTWA via Meta Marketing API com parâmetros do funil (Story 8.1).
6. Webhooks Meta configurados para receber notificações de status de mensagem.
7. Prioridade Should: depende de conta Meta Business + WABA verificada (PRD §6).

## Dev Notes

- PRD §5: "Meta Marketing API + WhatsApp Cloud API + webhooks".
- PRD F4.4: "integração Meta/WhatsApp API".
- PRD §6: pré-requisito "conta Meta Business + WABA verificada".
