## Story

**As a** empreendedor gerenciando minha carteira,
**I want** um guardrail que garanta investimento mínimo em cada pilar de monetização,
**so that** minha carteira mantenha diversificação entre Ads, Afiliações e Parcerias.

## Acceptance Criteria

1. Configuração de % mínimo por pilar disponível na UI de alocação (ex: mín. 10% em cada pilar).
2. Engine de alocação verifica se cada pilar atinge o mínimo após distribuição inicial.
3. Se pilar não atinge o mínimo, engine realoca verba para garantir o mínimo (deducindo de pilares acima do mínimo).
4. Alerta visual exibido se não for possível satisfazer todos os mínimos com o orçamento disponível.
5. Mínimos são opcionais (padrão = 0% por pilar); usuário pode desativar.
6. Restrição de mínimo é salva em `allocations.risk_band_ratios` como campos adicionais (ou tabela de configuração).
