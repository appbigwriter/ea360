## Story

**As a** usuário do EA360,
**I want** o catálogo GOM com ~50 canais distribuídos nos 3 pilares já populado no banco de dados,
**so that** o alocador e o matchmaking tenham dados reais para calcular scores e recomendar canais desde o primeiro uso.

## Acceptance Criteria

1. 3 pilares inseridos em `gom_pillars`: Ads, Afiliações, Parcerias.
2. Categorias inseridas em `gom_categories` conforme agrupamentos do guia `Guia-de-Opcoes-de-Advertising-EA360.md`.
3. Todos os ~50 canais inseridos em `gom_channels` com notas 1–5 para cada um dos cinco atributos (Custo, Payback, Controle, Risco, Escala).
4. Nenhum canal com valores NULL nos cinco score columns.
5. Seed implementado como migração SQL (`supabase/migrations/0006_gom_seed.sql`) para ser reproduzível.
6. `SELECT COUNT(*) FROM gom_channels` retorna >= 45.
