## Story

**As a** empreendedor na Entrevista 360,
**I want** que o sistema gere perguntas de aprofundamento personalizadas com base nas minhas respostas anteriores,
**so that** a entrevista seja adaptativa e capture nuances específicas do meu negócio.

## Acceptance Criteria

1. Após cada resposta do usuário, o sistema chama a API Anthropic para gerar 0–3 perguntas de follow-up baseadas no contexto acumulado.
2. Perguntas geradas são salvas em `interview_questions` com `parent_question_id` apontando para a pergunta-mãe.
3. A chamada ao LLM inclui: camada atual, perguntas anteriores e respostas como contexto.
4. O sistema é tolerante a falhas do LLM: se a chamada falhar, avança com as perguntas estáticas seguintes sem bloquear o usuário.
5. Indicador visual mostra quando o sistema está "pensando" (gerando próximas perguntas).
6. Chave `ANTHROPIC_API_KEY` usada apenas no servidor (Server Action / Route Handler); nunca exposta no cliente.
7. Perguntas geradas são apresentadas na UI após serem salvas (fluxo completo).
