## Acceptance Criteria

1. Projeto Next.js inicializado com App Router e Server Components habilitados (`next@15`, `react`, `react-dom`).
2. Tailwind CSS configurado com PostCSS e Autoprefixer; arquivo `tailwind.config.ts` presente.
3. shadcn/ui inicializado; ao menos um componente Button importável sem erros.
4. Dependências UI instaladas: `@radix-ui/*`, `lucide-react`, `cmdk`, `framer-motion`.
5. TypeScript configurado (`tsconfig.json`) com strict mode.
6. `npm run dev` sobe o servidor em `localhost:3000` sem erros.
7. `npm run build` conclui sem erros de tipagem ou lint.
8. Arquivo `.env.example` presente com variáveis mínimas documentadas (Supabase URL/KEY, Anthropic key).
